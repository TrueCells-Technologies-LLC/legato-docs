NAME

instsys - install system on target

SYNOPSIS

instsys SYSTEM_FILE [TARGET_IP]

DESCRIPTION

Install the given system on the target at TARGET_IP SYSTEM_FILE is the file containing the system(collection of apps) to be installed. E.g., 'mySys.ar7_sys'.

ENVIRONMENT

If the TARGET_IP value is not given and the environment variable DEST_IP
is set then DEST_IP will be used as TARGET_IP
